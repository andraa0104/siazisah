# SI-AZISAH Vue.js Frontend

Frontend Vue.js untuk Sistem Informasi Zakat dengan UI responsive.

## Setup

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```

## Akses

Development: `http://localhost:3000`
